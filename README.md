# MammaMia! Pizzeria

An interactive web application for an Italian pizzeria featuring online ordering, table booking, and a responsive home page.

## Features

- **Home Page** - Landing page with image carousel (Flickity), photo gallery with hover effects, and quick navigation to ordering and booking
- **Online Ordering** - Browse the menu, customize products with various options (toppings, sauces, crusts), adjust quantities, and place orders via a shopping cart
- **Table Booking** - Reserve tables using an interactive floor plan with date/time picker, real-time availability checking, and booking confirmation

## Tech Stack

- Vanilla JavaScript (ES6 modules)
- Handlebars templating
- SCSS (BEM methodology)
- Flickity carousel
- Flatpickr date picker
- JSON Server (REST API backend)
